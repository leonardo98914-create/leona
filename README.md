# Página Web de Motos

Este proyecto es una página web dedicada a mostrar opciones de motocicletas para que los usuarios puedan explorarlas y comprarlas.

##  Objetivo

El objetivo del sitio es permitir a los usuarios:

* Ver opciones modelos de motos.
* Conocer características y especificaciones.
* Comparar opciones antes de comprar.
* Acceder a una posible compra o contacto.

##  Tecnologías utilizadas

* HTML5
* CSS3
* Visual Studio (entorno de desarrollo)

##  Estructura del proyecto

El proyecto incluye:

* Páginas con información de cada moto.
* Imágenes de referencia.
* Menú de navegación para explorar las opciones.
* Sección de compra o contacto.

##  Cómo ejecutar el proyecto

1. Clonar o descargar el proyecto.
2. Abrir la carpeta en Visual Studio.
3. Ejecutar el archivo `index.html` desde el navegador o con Live Server.
